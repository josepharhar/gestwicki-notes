K18 HSQC:
- GOAL: rerun K18 apo HSQC at 285K to result in spectra that are more similar to Rebecca’s or more easily assignable using the published peak assignments; run K18 + DJA2, DJB1, DJB4, and His-tagged EB3
- see attached for chaperone/K18 dilution calculations
- some notes on data collection: 
  • no need to retune for successive experiments
  • if after standard shimming procedure final B0 stdev is >0.2, can add Z6 (by checking the box) for 1D shimming (as long as Z6 changes within 2000-10000 it’s ok)
  • performed 4 scans and 64 dummy scans
  • delay time = 1.5
  • use “edc” in command line to start a new experiment using the same acquisition parameters
- experiment identities: K18_apo_4 is the full collection (earlier experiments were attempts that did not include the full number of scans/dummy scans/etc); K18_DJB1_1 is a rerun of DJA2 (samples were not switched); K18_DJB1_2 is DJB1; 
- some notes on data processing:
  • procpars: set SF to 800.1299207, 81.0764591 (from Fatima’s settings, may or may not be correct for the new console)
  • SSB: changed from 2 to 2.5
  • perform phasing by .ph, right click on peak, add, and then use 0th order to adjust rows and save; xfb to reprocess
  • after phasing, save a parameter file: wpar 2D_K18_proc then select proc
  • to apply a saved parameter file to process a different spectrum: rpar 2D_K18_proc proc; xfb
  • once data has been processed in topspin, can open these files in Analysis and analyze as described previously
- RESULTS: 
  1. K18 apo spectrum acquired at 285K was much more similar to published peak assignments than spectrum acquired at 298K, although not all peaks overlapped with assignments
  2. K18 apo spectrum was similar to Rebecca’s K18 apo spectrum acquired under the same buffer conditions, but was not perfectly aligned, and some peaks were drastically different between the spectra
  3. all three spectra in the presence of chaperones were shifted in similar ways, by a quick visual inspection; some peaks that were present for chaperones but not apo were also present in Rebecca’s apo spectrum (see attached)
